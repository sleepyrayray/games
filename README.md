## Projects

### Who's the Spy?

Page: [https://sleepyrayray.github.io/games/whosthespy/](https://sleepyrayray.github.io/games/whosthespy/)

A mobile-first local multiplayer social deduction game for 3 to 10 players. Everyone shares one device, receives a secret word, gives spoken hints in real life, and tries to figure out who the spy is before the final guess steals the round.

More game projects will be added here over time.
